Trabalhando Com Listas

Listas Ordenadas

OBS*:Podemos indicar o inicio de listas ordenadas usando *start="(Número o qual começa a lista)"

Para uma lista ordenada em algarismo romano MINÚSCULO usamos :
type="i"

  1. Acordar
  2. Tomar banho
  3. Tomar café
  4. Trabalhar
  5. Voltar pra casa
  6. Jantar
  7. Assistir ao jogo
  8. Dormir

Para uma lista ordenada em algarismo romano MAIÚSCULO usamos:
type="I"

  1. Acordar
  2. Tomar banho
  3. Tomar café
  4. Trabalhar
  5. Voltar pra casa
  6. Jantar
  7. Assistir ao jogo
  8. Dormir

Para uma lista ordenada em ordem alfabética MAIÚSCULA usamos:
type="A"

  1. Acordar
  2. Tomar banho
  3. Tomar café
  4. Trabalhar
  5. Voltar pra casa
  6. Jantar
  7. Assistir ao jogo
  8. Dormir

Para uma lista ordenada em ordem alfabética MINÚSCULA usamos:

Listas Não Ordenadas

Para uma lista com indicadores na forma de um quadrado preenchido usamos:
type="square"

  • Pão
  • Manteiga
  • Café
  • Leite
  • Sal
  • Doce

Para uma lista com indicadores na forma de um círculo vazio usamos:
type="circle"

  • Pão
  • Manteiga
  • Café
  • Leite
  • Sal
  • Doce

Para uma lista com indicadores na forma de um disco preenchido usamos
type="disk"

  • Pão
  • Manteiga
  • Café
  • Leite
  • Sal
  • Doce
